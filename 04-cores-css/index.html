Cores em CSS

CSS oferece várias formas de especificar cores, os principais são:

Cores por nomes

CSS tem 140 nomes de cores pré-definidos como "blue", "red", "green", etc.

Sintaxe: color: blue;

Limitação: Paleta limitada de cores e menos controle preciso

Cores em RGB(Red, Green, Blue)

Cada componente (vermelho, verde, azul) varia de 0 a 255.

Sintaxe: color: rgb(255, 0, 0); (vermelho puro)

Vantagem: Mais controle sobre as cores.

Cores em RGBA(Red, Green, Blue, Alpha)

Igual ao RGB, mas com canal alpha (transparência) de 0 (transparente) a 1 (opaco).

Sintaxe: color: rgba(255, 0, 255, 0.5); (roxo com 50% de opacidade)

Uso comum: Efeitos de transparência e sobreposições.

Cores em Hexadeximal

Código de 6 dígitos hexadecimais (RRGGBB), onde #000000 é preto e #ffffff é branco.

Sintaxe: color: #ff00ff; (magenta)

Vantagens: Amplamente usado, formato compacto, suporta versão abreviada (#f0f).
